# employee-training-tracker
web application for managing employee training records, certifications, and compliance tracking using Flask and SQLite.

## Features
- Employee registration
- Training record management
- Certification tracking
- Training expiration monitoring
- Search and filtering
- Responsive web interface
- SQLite database integration

  ## Tech Stack
- Python
- Flask
- SQLite
- HTML5
- CSS3
- JavaScript

  ## Installation

  Clone the repository:
  
  ```bash
  git clone https://github.com/caiovrsdev/employee-training-tracker.git
  cd employee-training-tracker

  Install dependencies:

  pip install -r requirements.txt

  Run the application

  python app.py

  Open:

  http://localhost:5000

## Background

This project was created to improve training record management and compliance tracking in a corporate environment. The goal was to replace manual processes with a centralized web application.

Note: This project was originally developed in Brazilian Portuguese. The interface and some labels are currently available only in Portuguese. Internationalization may be added in future versions.
